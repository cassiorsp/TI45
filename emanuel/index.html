O que é git? um controle de revisão de codigos ultilizado em todo mundo. O que é github? é um sistema que agrupa desenvolver um projeto, sendo ultilizado o compartilhamento entre os desenvolvedores e armazenando as informações dos projetos criados. Apresente 3 alternativas para o github. gitlab - é uma plataforma git e devops baseado em nuvens que ajuda os desenvolvedores a monitorar, testar e implantar seus codigos. gitea - auto-armazenamento que tenta ser fácil de manusear e que possui recursos similares ao GitHub. Bitbucket - Como pertence a Atlassian, o serviço é totalmente integrado com outros gerentes de projetos pertencentes à mesma empresa como Jira. Para que serve o commit do github? para registrar alterações em um ou mais arquivos no seu branch. Para que serve o push origin? para efetuar push das alterações locais para o repositório online. O que é pullrequest github? permitem que você proponha alterações nos projetos, forneça feedback e revisões e mescle alterações em projetos. Explique o que é versionamento? consiste em estratégias para gerenciar as diferentes versões de um código, de um sistema ou de um modelo. É uma forma de administrar as mudanças que são feitas e de garantir mais segurança na transição de uma versão para outra. Explique o que é fluxo de trabalho colaborativo github? conjunto de práticas recomendadas e técnicas de colaboração que permitem que as equipes trabalhem perfeitamente nos projetos. Ele é centrado no controle de versão, facilitando o rastreamento eficiente de alterações, depuração e integração contínua.
